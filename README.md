# spring-demo
`spring-demo`是基于`maven`构建的一个示例项目，用于验证和演示spring框架中的相关特性和功能，本项目将从最基本的maven项目开始，逐步实现spring的相关配置并且添加项目开发中常用的第三个类库。

新功能或者新特性的实现将通过`分支`的方式实现，`mater`分支中包含所有分支的内容，若仅对某一特性感兴趣，请参看对应的分支已经该特性的相关介绍文档（下文将给出这些特性的文档连接）
